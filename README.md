# Plataforma-de-Organiza-o-academica-para-universit-rios
Desenvolver uma aplicação web para auxiliar estudantes universitários na organização de tarefas, projetos e rotina acadêmica.

## Requisitos Funcionais
RF01: O sistema deve permitir cadastro de usuários.

RF02: O usuário deve conseguir criar tarefas.

RF03: O sistema deve permitir definir prazos.

RF06: O usuário poderá editar tarefas.

RF05: O sistema deve enviar notificações de vencimento.

RF06: O usuário poderá excluir tarefas.


## Requisitos Não Funcionais
RNF01: O sistema deve ser responsivo.

RNF02: O tempo de resposta deve ser inferior a 3 segundos.

RNF03: Os dados devem ser armazenados em banco de dados.

RNF04: A interface deve seguir princípios de acessibilidade.

RNF05: O sistema deve possuir autenticação segura.

## Tecnologias
React

Node.js

Postgres
